# AUTO543305-电子线路设计设计训练专题实验2

## 课程介绍

电子线路设计的两个专题实验都是大三上学期上的

每个实验 24 个学时，0.75 个学分

| 课次 | 时间 | 内容 | 验收 | 报告 |
| ---- | ---- | --- | --- | ---- |
| 实验1 | 第 8~10 周 | 第一次做电子相册或秒表，两个其实差不多，都是合并例程用上各个功能。第二三次做密码锁、计算器或游戏机，我做的计算器 | 第一次和第二三次都要验收 | 第二三次写报告 |
| 实验2 | 第 11~13周 | 模拟直升机控制系统或温度控制系统，都做的直升机，老师说温度控制难调 | 需要验收 | 需要写报告 | 

## 文件解读

**工程**里**实验1**的两个压缩包，其中**秒表**是第一次的内容，**计算器**是第二三次的内容
