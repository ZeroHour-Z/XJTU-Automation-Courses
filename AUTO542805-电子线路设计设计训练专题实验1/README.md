# AUTO542805-电子线路设计设计训练专题实验1

## 课程介绍

电子线路设计的两个专题实验都是大三上学期上的

每个实验 24 个学时，0.75 个学分

| 课次 | 时间 | 内容 | 验收 | 报告 |
| ---- | ---- | --- | --- | ---- |
| 实验1 | 第 1~2 周 | 用 Altium Designer 画原理图和 PCB 图 | 验收正弦波发生器电路和智能控制器 | 智能控制器 |
| 实验2 | 第 3 周 | Proteus 仿真软件的使用 | 正弦波发生器电路、方波和三角波信号发生器电路、门铃电路 | 无 |
| 实验3 | 第 5~7周 | Proteus 和 Keil C51的联合使用 | 参考例程里选一个（计算器、密码锁、电子琴、游戏机、温度计），我的是密码锁 | 需要写报告 |

实验二不用交报告，只需要课上验收
