# XJTU-Automation-Courses

## 内容

西安交通大学本科自动化专业课程资料

本仓库内容是自动化大类 23 级课程资料，主要包含专业课和实验课内容，23 级培养方案相较与往年有较大变动，而课程资料对后人来说至少也有一年的时效空期，因此很多学长的课程资料都已落灰，于是我写了这个

每个仓库都写了 `README.md` 文件对于课程和文件进行说明

由于我写这个仓库的时候已经是大三下学期了，很多之前的课程详细内容我已经忘了，不过对于我还留有记忆的课程我会做更详细的说明
